# Destaca AI — Base de Conhecimento

> **"O modo de ferramenta em horas — o sistema fica."**

Repositório central do **Destaca AI**. Funciona como o **Segundo Cérebro** da operação — todos os agentes de IA (OpenClaw, Claude Desktop, GPT, etc.) leem e escrevem aqui.

---

## Estrutura

```
/
├── README.md              ← Você está aqui
├── AGENTS.md              ← Comportamento base dos agentes
├── SOUL.md                ← Persona/identidade
├── IDENTITY.md            ← Identidade do assistente
├── USER.md                ← Perfil do humano (Paulo Rocha)
├── TOOLS.md               ← Ferramentas e configurações locais
├── HEARTBEAT.md           ← Tarefas periódicas
├── .gitignore             ← Exclusões de segurança
│
├── agentes/               ← Definições de agentes (OpenClaw, Claude, GPT)
├── skills/                ← Skills especializadas (1 por função)
├── kanban/                ← Planejamento e tarefas
├── docs/                  ← Documentação do projeto
├── integracoes/           ← Configs de integrações (Meta, DSH, etc.)
├── artigos/               ← Artigos e conteúdo
├── banco/                 ← Schemas e migrações do banco de dados
└── memory/                ← Memória de longo prazo e logs de sessão
```
